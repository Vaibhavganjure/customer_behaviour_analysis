📌 Overview

This project demonstrates an end-to-end data analytics workflow using real-world retail customer data. It covers data extraction, cleaning, analysis, visualization, and reporting—similar to tasks performed by data analysts in organizations.

The goal is to generate meaningful insights from customer shopping behavior and present them through interactive dashboards and reports.

📂 Dataset
Retail customer shopping dataset
Contains information such as:
Customer demographics (age, gender)
Purchase details (items, amount, discounts)
Review ratings and shopping trends
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning & EDA
SQL (PostgreSQL / MySQL / SQL Server) – Data querying & analysis
Power BI – Dashboard creation & visualization
Excel – Basic data handling & validation
Gamma – Presentation (PPT) creation
🔄 Project Steps
1. Data Loading
Imported dataset into Python using Pandas
Checked structure, columns, and data types
2. Data Cleaning
Handled missing values (NaN)
Removed duplicates
Standardized column formats
3. Exploratory Data Analysis (EDA)
Analyzed customer purchase behavior
Identified trends using visualizations
Found key metrics like average purchase, top products
4. SQL Analysis
Loaded cleaned data into database
Performed queries such as:
Total revenue by category
Customer segmentation
Discount impact analysis
5. Dashboard Creation (Power BI)
Built interactive dashboards
Used KPIs like:
Total Sales
Average Order Value
Customer Distribution
Added filters and slicers for better insights
6. Reporting & Presentation
Created a structured report
Built a presentation (PPT) using Gamma
Highlighted insights and business recommendations
📊 Dashboard Highlights
Sales performance overview
Customer segmentation analysis
Product-wise revenue breakdown
Discount vs purchase behavior insights
📈 Results & Insights
Identified high-value customer segments
Found top-selling products and categories
Observed impact of discounts on revenue
Provided data-driven recommendations for business growth
